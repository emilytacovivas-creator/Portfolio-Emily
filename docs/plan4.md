Prompt: "El scroll entre secciones es muy brusco. ¿Cómo puedo implementar un scroll suave (smooth scroll) y asegurar que el menú de navegación resalte la sección donde se encuentra el usuario?"

Respuesta de la IA:

Integrar la librería Lenis para el suavizado del movimiento.

Crear una función updateActiveNav vinculada a ScrollTrigger para detectar la sección activa.

Añadir scroll-margin-top en CSS para que las secciones no queden tapadas por el Navbar fijo.

Notas de implementación: Se mejoró la navegación y se añadió el efecto de subrayado dinámico en los enlaces del menú.
