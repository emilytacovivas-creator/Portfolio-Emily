# Portfolio - Emily Taco

> by Emily Taco

## Descripción

Portfolio personal desarrollado para la asignatura de programación. El proyecto destaca por un diseño minimalista, transiciones fluidas y un enfoque orientado a la experiencia de usuario (UX).

## Stack Tecnológico

- **HTML5 Semántico** y CSS3 (Diseño Responsive)
- **Bootstrap 5.3** (Sistema de rejilla y utilidades)
- **GSAP** (ScrollTrigger, Flip, ScrollTo para animaciones avanzadas)
- **Lenis** (Smooth Scroll para una navegación suave)

## Política de Uso de IA

Se ha utilizado IA (Gemini/ChatGPT) como asistente de planificación técnica y auditoría de requisitos. El proceso seguido es:

1.  **Fase de Planificación:** Documentada en la carpeta `/docs` con archivos individuales por tarea.
2.  **Fase de Implementación:** Código aplicado tras la validación del plan y documentado con mensajes de commit específicos.

## Configuración y Personalización

### Desarrollo Local

1. Clonar el repositorio: `git clone https://github.com/tu-usuario/nombre-repo.git`
2. Abrir el proyecto en VS Code.
3. Lanzar la extensión **Live Server** para visualizar en tiempo real.

### Guía de Personalización

- **Colores:** Modificar las variables en el archivo CSS dentro de `assets/css/`.
- **Contenido:** Los textos de las secciones se encuentran directamente en `index.html`.
- **Imágenes:** Sustituir los enlaces de ImageKit en el HTML por tus propias URLs o archivos en `assets/images/`.
